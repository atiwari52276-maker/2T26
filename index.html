HTML BASICS

👉HTML (HyperText Markup Language) ek standard markup language hai jo web pages banane ke liye use hoti hai. Ye website ka structure define karti hai, jaise headings, paragraphs, images aur links. HTML browser ko batata hai ki content ko kaise display karna hai. 👈

HTML एक मार्कअप लैंग्वेज है जिसका उपयोग वेबसाइट का ढांचा (structure) बनाने के लिए किया जाता है। यह बताता है कि वेब पेज में heading, paragraph, image, link कहाँ होंगे। HTML ब्राउज़र को निर्देश देता है कि content कैसे दिखाना है।

Css Basics

👉CSS (Cascading Style Sheets) ek styling language hai jo HTML pages ko design karne ke liye use hoti hai. Ye website ka color, layout, font aur spacing control karti hai. CSS se web page attractive aur responsive banaya jata hai.👈

CSS एक स्टाइलिंग लैंग्वेज है जो वेबसाइट को सुंदर और आकर्षक बनाती है। इससे हम color, font, layout, spacing को कंट्रोल करते हैं। CSS की मदद से वेबसाइट responsive और design वाली बनती है।

JavaScript

👉JavaScript ek programming language hai jo website me interactivity add karti hai. Isse button click, form validation, animations jaise kaam hote hain. JavaScript se web page dynamic aur smart ban jata hai👈

JavaScript एक प्रोग्रामिंग लैंग्वेज है जो वेबसाइट में interactivity जोड़ती है। इससे button click, form validation, animation जैसे काम होते हैं। JavaScript वेबसाइट को dynamic और smart बनाता है।
